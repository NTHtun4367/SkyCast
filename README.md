# SkyCast 🌤️

**SkyCast** is a high-performance, secure, and seamless weather application built with **React Native**. Leveraging **NativeWind** for utility-first styling, SkyCast provides users with real-time atmospheric data and precision forecasts wrapped in a modern, minimalist interface.

-----

## ✨ Features

  * **Real-Time Data:** Instant access to current temperature, humidity, and wind conditions.
  * **7-Day Forecast:** Detailed daily and hourly projections for the upcoming week.
  * **Global Search:** Dynamic location searching for weather updates anywhere in the world.
  * **Native Performance:** Optimized for fluid animations and fast load times on both iOS and Android.
  * **Responsive UI:** A clean, dark-mode inspired design built using NativeWind (Tailwind CSS).

-----

## 🛠️ Tech Stack

  * **Framework:** [React Native](https://reactnative.dev/) (Expo)
  * **Styling:** [NativeWind](https://www.nativewind.dev/) (Tailwind CSS)
  * **Data Source:** [Weather API](https://www.weatherapi.com/)
  * **Icons:** [@expo/vector-icons](https://docs.expo.dev/guides/icons/) v15.0.3
